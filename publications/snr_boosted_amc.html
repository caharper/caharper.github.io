                        In my work, SNR-Boosted Automatic Modulation Classification, we achieved state-of-the-art performance on the large-scale RadioML 2018.01A dataset, at the time. We have since improved upon our work (see AMC with Deep Neural Networks). In this work, we utilize a two-stage classification pipeline by first estimating the signal-to-noise ratio (SNR) of the input signal. Based on the estimated SNR, we classify the modulation type using a fine-tuned classification model for signals within the estimated SNR range.

                        # Paper Purpose

                        High performing automatic modulation classification (AMC) is crucial in modern communication systems. The purpose of this work is to improve the performance of AMC systems by incorporating signal-to-noise ratio (SNR) estimation. We demonstrate that by estimating the SNR of the input signal, we can improve the classification accuracy of the modulation type. This work is a stepping stone to our more recent work, AMC with Deep Neural Networks, where we further improve the performance of AMC systems by utilizing deep learning architectures.

                        # Methodology

                        Often observed in AMC systems, the classification accuracy of the modulation type decreases as the signal-to-noise ratio (SNR) decreases. In other words, as the signal becomes more noisy, the classification model has a harder time distinguishing between modulation types. In a prior workopen in new window of ours, we also found this to be the case (see below).

                        Most methods for AMC utilize a single classification architecture. Looking at the performance curves in the above figure, it is clear AMC performance is highly dependent on the SNR of the input signal. What if we could have a different classification model that is trained specifically for each SNR range? This is the motivation behind our work.

                        Although we cannot know the true SNR of the input signal, we can estimate it. We propose a two-stage classification pipeline. In the first stage, we estimate the SNR of the input signal. In the second stage, we classify the modulation type using a fine-tuned classification model for signals within the estimated SNR range. Below, you can see the general overview of our proposed method.

                        Since performance plateaus in performance below -8 dB and above 8 dB, we group (−∞, -8) and (8, ∞) into their own ranges. We then train a separate classification model for each SNR range. This allows us to achieve state-of-the-art performance on the RadioML 2018.01A dataset (at the time). Below, you can see the performance improvement of our SNR-boosted AMC system compared to the previous state-of-the-art system.

                        While the percentage improvement may seem small, it is important to note that these results are statistically significant at a 0.5 p-value threshold on a McNemar's Testopen in new window. Additionally, given the large scale of the RadioML 2018.01A test set (1 million samples), even small percentage improvements can have a large impact in practice.
